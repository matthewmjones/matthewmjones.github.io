Topic 3 Integration

3.1 Understanding the integral

Let \(f\) be a function and suppose for the moment that \(f(x) \ge 0\) for all \(x\). The integral of \(f\) between to numbers \(a\) and \(b\) with \(a < b\) is the area between the graph \(G_f\) and the \(x\)-axis from \(x=a\) up to \(x=b\). It is written as \[ \int_a^b f(x) \mathrm{d}x.\]

The fundamental theorem of calculus gives us an easy way to find this area. We’ll prove this at the end of the year, but for now we’ll just write it informally.

If \(F\) is a function such that \(f(x) = F'(x)\) then \[ \int_a^b f(x) \mathrm{d}x = F(b) - F(a).\] The function \(F\) is called the of \(f\), or sometimes the anti-derivative. It is also often written as \[ \int f(x)\mathrm{d}x,\] that is, as an integral without the limits of integration.

Figure 1.1: The integral of a function between \(x=a\) and \(x=b\)

Example

Calculate: \[ \int_0^2 2x \;\mathrm{d}x. \] Here we may take \(F(x) = x^2\). Then \(F'(x) = 2x = f(x)\) and we have \[ \int_0^2 2x \;\mathrm{d}x = \left[ x^2 \right]_0^2 = 4. \]

Because we are interested in finding anti-derivatives, integration is often seen as the opposite operation to differentiation. For this reason a great deal of effort is put into finding anti-derivatives. In the formula sheet you downloaded previously there are a number of standard anti-derivatives.

There are also rules of integration equivalent to the product and chain rules of differentiation.

3.2 Integration by substitution

The opposite operation to the chain rule is integration by substitution. This is more of an art than a science since you need to know what substitution to make.

Example

Find the indefinite integral \[ \int x^2(x^3 - 3)^5\mathrm{d}x.\] Here we could expand this bracket and multiply each term by \(x^2\). But a better method would be to notice that the derivative of \((x^3 - 3)\) is \(3x^2\) which is a multiple of \(x^2\). So we would expect the indefinite integral to be of the form \(k(x^3 - 3)^6\). If we differentiate this we get \[ 18kx^2 (x^3 - 3)^5, \] so that we must have \(k=1/18\). Hence \[ \int x^2(x^3 - 3)^5\mathrm{d}x = \dfrac{1}{18} (x^3 - 3)^6 + c.\]

You don’t always see this composition so a more general method needs to be used.

Let \(F(x) = \int f(x)\mathrm{d}x\) so that \(F'(x) = f(x)\). Suppose the variable \(x\) is itself a function of another variable \(u\), \(x = g(u)\). This means \[ f(x) = f(g(u)). \] This is a function of the new variable \(u\) so we can use the chain rule to find \[ f'(u) = g'(u)\, f'(g(u)) = g'(u) f'(x).\] This may seem a little cyclic but the point is that we can write \[ f'(x) = \dfrac{f'(u)}{g'(u)}.\] Writing \(g'(u) = \dfrac{\mathrm{d}x}{\mathrm{d}u}\) this means we have the easier to remember formula \[ \int f(x)\mathrm{d}x = \int f(g(u)) \dfrac{\mathrm{d}x}{\mathrm{d}u} \mathrm{d}u.\]

Example

Find the indefinite integral \[ \int x^2(x^3 - 3)^5\mathrm{d}x.\] This time we will answer this question using the substitution \(u = x^3 - 3\). Now \(\dfrac{\mathrm{d}u}{\mathrm{d}x} = 3x^2\) and so \(\dfrac{\mathrm{d}x}{\mathrm{d}u} = \dfrac{1}{3x^2}\). So \[ \begin{aligned} \int x^2(x^3 - 3)^5\mathrm{d}x &= \int x^2 u^5 \dfrac{1}{3x^2} \mathrm{d}u\\ &= \int \dfrac{u^5}3 \mathrm{d}u\\ &= \dfrac{u^6}{18} + c\\ &= \dfrac{1}{18} (x^3 - 3)^6 + c \end{aligned} \] which is what we found before.

Note in some examples we make the substitution \(u = G(x)\) instead of \(x = g(u)\). Then we can use the fact that as long as \(\dfrac{\mathrm{d}u}{\mathrm{d}x} \neq 0\), \[ \dfrac{\mathrm{d}x}{\mathrm{d}u} = \dfrac{1}{\dfrac{\mathrm{d}u}{\mathrm{d}x}}.\] Note in some examples we make the substitution \(u = G(x)\) instead of \(x = g(u)\). Then we can use the fact that as long as \(\dfrac{\mathrm{d}u}{\mathrm{d}x} \neq 0\), \[ \dfrac{\mathrm{d}x}{\mathrm{d}u} = \dfrac{1}{\dfrac{\mathrm{d}u}{\mathrm{d}x}}.\] Note: this is a gross simplification of something called the inverse mapping theorem but it will do for now.

Example

Find the indefinite integral \[ \int \sin \frac{x}3 \mathrm{d}x.\] This time we let \(u = \dfrac{x}3\) so that \(\dfrac{\mathrm{d}u}{\mathrm{d}x} = \dfrac{1}3\). Therefore \[ \begin{aligned} \int \sin \frac{x}3 \mathrm{d}x &= \int (\sin u) 3 \mathrm{d}u\\ &= -3\cos u + c \\ &= -3 \cos \dfrac{x}3 + c \end{aligned} \]

Suppose now we want to integrate a function \[ \int_a^b f(x) \mathrm{d}x\] using a suitable substitution \(x = g(u)\). How should we go about this? First we have that \(x = g(u)\) is injective — i.e. it is a one-to-one mapping. That way we can find values of \(u\) so that \[a = g(A), \qquad b= g(B).\] Note this just means \(A = g^{-1}(a)\) and \(B = g^{-1}(b)\). Then \[ \int_a^b f(x)\mathrm{d}x = \int_A^B f(g(u)) \dfrac{\mathrm{d}x}{\mathrm{d}u} \mathrm{d}u.\]

Example

Calculate the integral \[ \int_0^1 \sqrt{1-x^2} \mathrm{d}x.\] For integrals like this we can use the substitution \(x = \sin u\). Then \(\dfrac{\mathrm{d}x}{\mathrm{d}u} = \cos u\). Now when \(x = 0\), \(u = 0\) and when \(x=1\), \(u=\pi/2\). So \[ \begin{aligned} \int_0^1 \sqrt{1-x^2} \mathrm{d}x &= \int_0^{\pi/2} \sqrt{1-\sin^2 u} \cos u \mathrm{d}u\\ &= \int_0^{\pi/2} \sqrt{\cos^2 u} \cos u \mathrm{d}u\\ &= \int_0^{\pi/2} \cos^2 u \mathrm{d}u\\ &= \int_0^{\pi/2} \frac{1}2(\cos 2u + 1) \mathrm{d}u\\ &= \left[ \dfrac{1}2 (\frac{1}2 \sin 2u + u) \right]_0^{\pi/2}\\ &= \dfrac{\pi}4. \end{aligned} \]

3.3 Integration by parts

Suppose \(F(x)\) is the indefinite integral of \(f\), i.e. \(F'(x) = f(x)\). The product rule for differentiation said that if \(F = gh\) then \[ F'(x) = g'(x)h(x) + g(x)h'(x).\] This means that to find \(F(x)\) we can integrate the right-hand side of this. \[ F(x) = \int f(x) \mathrm{d}x = \int g'(x)h(x)\mathrm{d}x + \int g(x)h'(x)\mathrm{d}x.\] This gives us another useful method for finding indefinite integrals called integration by parts. We normally write this a \[ \int g'(x) h(x) \mathrm{d}x = g(x) h(x) - \int g(x)h'(x)\mathrm{d}x. \] This method is useful when we want to integrate a product.

Example

Find \[ \int x e^{x} \mathrm{d}x.\] We write \(g'(x) = e^x\) here, and \(h(x) = x\). Then \[ g(x) = e^x, \qquad h'(x) = 1.\] Therefore \[ \begin{aligned} \int g'(x) h(x) \mathrm{d}x &= \int x e^{x} \mathrm{d}x \\ &= xe^x - \int e^x \mathrm{d}x\\ &= xe^x - e^x + c = (x-1)e^x + c \end{aligned} \]

Example

Calculate \[ \int_0^{\pi/2} x \cos x \mathrm{d}x. \] Here we let \(g'(x) = \cos x\) and \(h(x) = x\). Then \[ g(x) = \sin x,\qquad h'(x) = 1.\] We can therefore calculate the indefinite integral \[ \int x \cos x \mathrm{d}x = x\sin x - \int \sin x \mathrm{d}x = x\sin x + \cos x + c.\] Hence

\[ \begin{aligned} \int_0^{\pi/2} x \cos x \mathrm{d}x &= \left[ x\sin x + \cos x\right]_0^{\pi/2}\\ &= \left(\frac{\pi}2 + 0\right) - \left(0 + 1\right)\\ &= \frac{\pi}2 - 1. \end{aligned} \]

3.4 Pattern recognition

The more practice you have doing integration the better you will become. In addition to the techniques above there are often patterns that come up a lot. For example we have the logarithmic integral: \[ \int \frac{f'(x)}{f(x)} \mathrm{d} x = \log |f(x)| + c\]

Example

Find \(\displaystyle \int \frac{x}{1+x^2} \mathrm{d}x\).

Here we let \(f(x) = 1 + x^2\). Then \(f'(x) = 2x\) and we have \[ \int \frac{x}{1+x^2} \mathrm{d}x = \frac{1}2\int \frac{2x}{1+x^2} \mathrm{d}x = \frac{1}2 \log |1 + x^2| + c.\] Note: \(1 + x^2 > 0\) for all \(x\) so we can remove the absolute value sign here. \[ \int \frac{x}{1+x^2} \mathrm{d}x = \frac{1}2 \log (1 + x^2) + c\]

Example

Find \(\displaystyle \int \tan x \mathrm{d}x\).

Here we write \(\displaystyle \tan x = \frac{\sin x}{\cos x}\). Then \[ \int \tan x \mathrm{d}x = \int \frac{\sin x}{\cos x}\mathrm{d}x = -\log |\cos x| + c. \] Since \(\sec x = 1/\cos x\) we can write this as \[ \int \tan x \mathrm{d}x = \log |\sec x| + c.\]

Other techniques that you will have learned include:

  1. Partial fractions
  2. Polynomial long-division
  3. Trigonometric identities. These include the double angle formulas, the *half angle formulas** as well as Pythagoras’ identity, \[ \sin^2 x + \cos^2 x = 1.\]

3.5 Areas under a curve

The integral of \(f\) measures the area between the graph of a curve, \(G_f\), and the \(x\)-axis only if \(f(x)\ge 0\) for all \(x\). If \(f(x) < 0\) at any point then the integral gives the negative value of the area.

Example

Calculate the integral \[ \int_{-1}^1 x^2 - 1 \mathrm{d}x.\] The answer is \(-4/3\) (Check that). This is negative because the graph of this curve is beneath the \(x\)-axis.

This phenomenon can lead to cancelling out in integrals.

Example

Calculate the integral \[ \int_0^{2\pi} \sin x \mathrm{d}x.\] This time we have \[ \int_0^{2\pi} \sin x \mathrm{d}x = \left[ -\cos x\right]_0^{2\pi} = 0 - 0 = 0.\] The graph consists of pieces above the \(x\)-axis and pieces below, their areas cancel out.

An odd function is any function that satisfies \[ f(-x) = -f(x). \]

Example

Suppose \(f\) is an odd function. Then \[ \int_{-A}^A f(x) \mathrm{d}x = 0,\] for any \(A > 0\).

This is true because odd functions are rotationally symmetric by \(\pi\) radians around the origin. This means the graph for \(x<0\) is the same as the graph for \(x>0\) except that it is beneath the \(y\)-axis.

We can prove this more rigorously as follows. Making the substitution \(u = -x\), \[\begin{aligned} \int_{-A}^0 f(x) \mathrm{d}x &= \int_{A}^0 -f(-u) \mathrm{d}u \\ &= \int_A^0 f(u) \mathrm{d}u = - \int_0^A f(u) \mathrm{d}u\\ &= - \int_0^A f(x) \mathrm{d}x \end{aligned} \] So, \[ \int_{-A}^A f(x) \mathrm{d}x = \int_{-A}^0 f(x) \mathrm{d}x + \int_{0}^A f(x) \mathrm{d}x = -\int_0^A f(x) \mathrm{d}x + \int_0^A f(x) \mathrm{d}x = 0.\]
