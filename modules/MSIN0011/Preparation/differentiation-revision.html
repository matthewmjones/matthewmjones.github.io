Topic 1 Differentiation Revision

1.1 Understanding the derivative

The derivative of a function \(f(x)\) is the gradient of the tangent to its graph at the point \((x,f(x))\). Figure @ref(gradient plot) illustrates this and we will look in detail at this when we learn about differentiable functions in Week 2.

1.2 The gradient

The gradient is a number that measure how quickly a function \(f(x)\) changes per unit increase in \(x\). For a straight line this is straightforward.

Let \(f(x) = mx + c\) for some numbers \(m,c\in\mathbb{R}\). When \(x\) increases to from \(1\) to \(2\), say, the function \(f\) increases from \(m + c\) to \(2m + c\). In other words when we increase \(x\) by \(1\) the function increases by \[ (2m + c) - (m + c) = m. \] The value of \(m\) represents the amount the function changes when \(x\) increases by \(1\).

  • When \(m > 0\) the function \(f(x)\) is strictly increasing
  • When \(m < 0\) the function \(f(x)\) is strictly decreasing
  • When \(m=0\) the function \(f(x)\) is constant

For this example the gradient doesnâ€™t change if \(x\) does. This is not true in general.

The derivative measures the instantaneous gradient, meaning the gradient at that particular point. Of course the gradient can change when \(x\) does, but the characterisation above is the same (except you have to be careful when you say that a function is constant as we will see in Week 1).

1.3 Calculating derivatives

The derivatives of some standard functions are given as fixed formulas, for example if \(f(x) = x^n\) then its derivative is \(f'(x) = nx^{n-1}\). There are a number of other such rules that you can find on the following formula sheet. You can print this out on an A3 piece of paper and fold it (there is also a copy on Moodle).

1.4 Rules for differentiating

There are also rules for differentiation that help you find more complicated derivatives. First we note that the derivative of a linear combination of functions, such as \[af(x) + bg(x),\] is given by \[ a f'(x) + b g'(x).\]

Example

The derivative of \(x^2 + 2x + 1\) is \[2x+2 \]

1.5 The chain rule

If a function \(f\) can be written as the composition of two other function \(g\) and \(h\), \[ f = g\circ h\] then we may calculate the derivative using the chain rule: \[ f'(x) = h'(x)\, g'(h(x)).\]

Example

Find the derivative of \(f(x) = \log (\cos x)\).

Here \(g(x) = \log x\) so that \(g'(x) = 1/x\), and \(h(x) = \cos x\) so that \(h'(x) = -\sin x\). \[ f'(x) = -\sin x \dfrac{1}{\cos x} = - \tan x,\] since \(\tan x = \sin x / \cos x\).

1.6 The product rule

If a function \(f\) can be written as a product of two other functions \(g\) and \(h\), \[ f = g h\] then we may calculate the derivative using the product rule: \[ f'(x) = g'(x) h(x) + g(x) h'(x).\]

Example

Find the derivative of \(f(x) = x^2 \cos x\).

Here we let \(g(x) = x^2\) and \(h(x) = \cos x\) so that \[ f'(x) = 2x \cos x + x^2 (-\sin x) = x(2\cos x - x\sin x).\]

Example

Find the derivative of \(f(x) = e^x \sin 5x\).

Here we let \(g(x) = e^x\) and \(h(x) = \sin 5x\).

Now \(g'(x) = e^x\) and we can calculate, using the chain rule, that \(h'(x) = 5\cos 5x\). So \[ f'(x) = e^x \sin 5x + e^x (5\cos 5x) = e^x (\sin 5x + 5\cos 5x).\]

1.7 The quotient rule

If a function \(f\) can be written as a quotient of two other functions \(g\) and \(h\), \[ f = \frac{g}h\] then we may calculate the derivative using the quotient rule: \[ f'(x) = \dfrac{g'(x) h(x) - g(x) h'(x)}{h(x)^2}. \]

Example

Find the derivative of \(f(x) = \dfrac{e^x}{\cos x}\).

Here \(g(x) = e^x\), so that \(g'(x) = e^x\), and \(h(x) = \cos x\) so that \(h'(x) = -\sin x\). \[ f'(x) = \dfrac{e^x \cos x - e^x (-\sin x)}{(\cos x)^2} = e^x \dfrac{\cos x + \sin x}{\cos^2 x} = e^x(1 + \tan x)\sec x.\]

Example

Find the derivative of \(f(x) = \dfrac{x^2 \cos 2x}{\sin^2 x}\).

Let \(g(x) = x^2 \cos 2x\). Using the product rule and the chain rule we have that \[ g'(x) = 2x \cos 2x + x^2 (-2\sin 2x) = 2x(\cos 2x - x\sin 2x)\] Let \(h(x) = \sin^2 x\). using the chain rule we have that \[ h'(x) = 2\sin x \cos x = \sin 2x. \] Therefore, \[ \begin{aligned} f'(x) &= \dfrac{2x(\cos 2x - x\sin 2x) \sin^2 x - x^2\cos 2x (\sin 2x)}{\sin^4 x}\\ &= \dfrac{2x\cos 2x\sin^2 x - 2x^2\sin 2x\sin^2 x - x^2 \cos2x \sin 2x}{\sin^4 x} \end{aligned} \]
