Topic 4 Integration Practice Problems

4.1 Anti-derivative problems

Find the following anti-derivatives:

  1. \(\displaystyle\int (3x^2-6) \mathrm{d}x\)
  2. \(\displaystyle\int \sin 2x\mathrm{d}x\)
  3. \(\displaystyle\int \frac{2}{x^2-1} \mathrm{d}x\)
  4. \(\displaystyle\int 3e^x \mathrm{d}x\)
  5. \(\displaystyle\int e^{3x} \mathrm{d}x\)
  6. \(\displaystyle\int \frac{1}{x^2} \mathrm{d}x\)
  7. \(\displaystyle\int \frac{1}{x}\mathrm{d}x\)
  8. \(\displaystyle\int \cos^3 2x\mathrm{d}x\)
  9. \(\displaystyle\int \frac{2}{(2x-3)^3}\mathrm{d}x\)
  10. \(\displaystyle\int \frac{2}{2x+3}\mathrm{d}x\)
  11. \(\displaystyle\int \frac{1}{\sqrt{1-x^2}}\mathrm{d}x\)
  12. \(\displaystyle\int x(x-1)^6 \mathrm{d}x\)
  13. \(\displaystyle\int \frac{1}{4+x^2}\mathrm{d}x\)
  14. \(\displaystyle\int \frac{8x}{(x^2-4)^5}\mathrm{d}x\)
  15. \(\displaystyle\int 2\sin 5x\cos 4x\mathrm{d}x\)
  16. \(\displaystyle\int (x+3)(x+7)^5\mathrm{d}x\)
  17. \(\displaystyle\int \frac{1+x}{\sqrt{1-x^2}}\mathrm{d}x\)
  18. \(\displaystyle\int x\cos x\mathrm{d}x\)
  19. \(\displaystyle\int 4x(3x-5)\mathrm{d}x\)
  20. \(\displaystyle\int \frac{10}{25+x^2}\mathrm{d}x\)
  21. \(\displaystyle\int 9\cos 3x\mathrm{d}x\)
  22. \(\displaystyle\int x^3\log x\mathrm{d}x\)
  23. \(\displaystyle\int 2^x\mathrm{d}x\)
  24. \(\displaystyle\int \ln x\mathrm{d}x\)
  25. \(\displaystyle\int \frac{2x^3+7x^2+2x-10}{(x+3)(2x-1)}\)

4.2 Integration problems

Calculate the following integrals.

  1. \(\displaystyle \int_0^1 x^2 \mathrm{d}x\)
  2. \(\displaystyle \int_1^{e} \frac{1}x \mathrm{d}x\)
  3. \(\displaystyle \int_0^{2\pi} \cos x \mathrm{d}x\)
  4. \(\displaystyle \int_0^{1/2} \sin (2\pi x) \mathrm{d}x\)
  5. \(\displaystyle \int_0^{\pi/4} \sin^2 x \mathrm{d}x\)
  6. \(\displaystyle \int_0^{\pi/2} x\cos 2x \mathrm{d}x\)
  7. \(\displaystyle \int_{-\pi/4}^{\pi/4} \sec x \mathrm{d}x\)
  8. \(\displaystyle \int_{-2}^1 \frac{3x}{x^2+2x-8} \mathrm{d}x\)
  9. \(\displaystyle \int_0^1 x e^{x^2} \mathrm{d}x\)
  10. \(\displaystyle \int_0^1 x^2 e^x \mathrm{d}x\)
  11. \(\displaystyle \int_0^1 \frac{x^4(1-x)^4}{x^2+1} \mathrm{d}x\).
